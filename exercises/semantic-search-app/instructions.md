# Expanded Search Tool

**Module:** 7 — Applied AI: Embeddings & Retrieval
**Estimated time:** 40 minutes

## Objective

Extend the Guided Example's semantic search tool with source filtering, result counts, and expandable full text.

## What You'll Build

A Streamlit app backed by ChromaDB that searches a `docs/` folder. The sidebar includes a multiselect source filter. Each result shows a 150-character preview in an expander and the total count displays as "Showing X of Y total documents".

## Reference Code

The starter file (`starter.py`) provides a scaffold with TODOs — fill in each section, then compare with the solution.

## Running

```bash
streamlit run starter.py
```

## Deliverable

A running Streamlit search app with source filtering, result counts, and expandable previews. At least 5 documents in the `docs/` folder.
