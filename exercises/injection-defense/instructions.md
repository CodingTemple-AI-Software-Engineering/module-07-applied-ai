# Injection Defense Lab

**Module:** 7 — Applied AI: Embeddings & Retrieval
**Estimated time:** 25 minutes

## Objective

Build input validation and output checking functions that protect a RAG-powered application from common prompt injection attacks.

## What You'll Build

A script with an input validator (checks 8+ suspicious patterns), an output validator (checks for things that shouldn't appear in responses), and a safe system prompt template — all tested against at least 5 test cases.

## Reference Code

The starter file (`starter.py`) provides a scaffold with TODOs — fill in each section, then compare with the solution.

## Running

```bash
python starter.py
```

## Deliverable

A working `starter.py` with input/output validators and a safe system prompt template, demonstrated against a mix of safe inputs, direct injections, and edge cases.
