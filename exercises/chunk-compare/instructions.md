# Chunk and Compare

**Module:** 7 — Applied AI: Embeddings & Retrieval
**Estimated time:** 30 minutes

## Objective

Chunk a real document using two different strategies, embed both sets, and compare which strategy produces better search results for the same queries.

## What You'll Build

A script that takes a 500–800 word document, applies fixed-size chunking (300 chars, 50 overlap) and paragraph-based chunking, embeds both sets, runs 3 queries against each, and prints a side-by-side comparison with a written analysis.

## Reference Code

The starter file (`starter.py`) provides a scaffold with TODOs — fill in each section, then compare with the solution.

## Running

```bash
python starter.py
```

## Deliverable

A working script that demonstrates the practical impact of chunking strategy on search quality, with your written analysis in the output.
