# Course Content Search

**Module:** 7 — Applied AI: Embeddings & Retrieval
**Estimated time:** 25 minutes

## Objective

Build a simple semantic search tool over content from modules you've already completed, demonstrating how embeddings find relevant results by meaning rather than exact keywords.

## What You'll Build

An interactive command-line script that embeds a list of course-summary sentences, then lets you search them using natural language queries. Results are ranked by cosine similarity, and the loop continues until you type "quit".

## Reference Code

The starter file (`starter.py`) provides a scaffold with TODOs — fill in each section, then compare with the solution.

## Running

```bash
python starter.py
```

## Deliverable

A working `starter.py` that performs interactive semantic search over at least 10 course-summary sentences, displaying the top 3 results with similarity scores.
