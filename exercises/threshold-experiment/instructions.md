# Similarity Threshold Experiment

**Module:** 7 — Applied AI: Embeddings & Retrieval
**Estimated time:** 25 minutes

## Objective

Explore how different similarity thresholds affect search quality, building intuition for tuning search systems.

## What You'll Build

A script that embeds a 15-sentence knowledge base covering 3–4 distinct topics, runs 5 test queries at three different thresholds (0.3, 0.5, 0.7), and prints how many results pass each threshold along with which ones are included or missed.

## Reference Code

The starter file (`starter.py`) provides a scaffold with TODOs — fill in each section, then compare with the solution.

## Running

```bash
python starter.py
```

## Deliverable

A working script that demonstrates how threshold choice affects result quantity and relevance across 5 queries at 3 threshold levels.
