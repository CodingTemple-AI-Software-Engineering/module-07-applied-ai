# Prompt Workshop

**Module:** 7 — Applied AI: Embeddings & Retrieval
**Estimated time:** 25 minutes

## Objective

Practice writing engineered prompts using framing, specificity, and few-shot examples, and observe how prompt quality affects output quality.

## What You'll Build

A script that sends three pairs of "bad" and "good" prompts to an API (or mock function) and displays results side by side, demonstrating measurable improvement from prompt engineering techniques.

## Reference Code

The starter file (`starter.py`) provides a scaffold with TODOs — fill in each section, then compare with the solution.

## Running

```bash
python starter.py
```

## Deliverable

A working script demonstrating 3 bad/good prompt pairs with their outputs, plus a written system prompt for a Course Study Assistant.
